FAQ

Frequently Asked Questions
What is HTML?

HTML stands for Hypertext Markup Language. It is the stander markup language for creating web pages and web appilcations. It describes the structure of a web page.

What is CSS?

CSS, or Cascading Style Sheets, is a stylesheet language used to describe the presentation and styling of a document written in a markup language, most commonly HTML. It is a core technology of the World Wide Web, working alongside HTML and JavaScript to create web pages.

What is the main difference between HTML and CSS??

HTML (HyperTextMarkup Language) is a markup language used to structure the content of a web page, while CSS (Cascading Style Sheets) is used to style and visually lay out the content on a web page. HTML provides the structure and content of a web page, while CSS provides the visual design.

Why do we put CSS in a separate file?

Using a separate file for CSS is a web development best practice that improves organization, maintainability, reusability, and website performance. This method is known as "external styling" and is used on almost all professional websites.

What is the difference between an H1 and a P tag?

The difference between an h1 and a p tag lies in their semantic meaning and function, not just their appearance. The h1 tag defines the most important heading on a page, while the p tag defines a paragraph of text.

What does the 'alt' attribute do in an img tag?

The alt (alternative text) attribute in an img tag provides a text description of an image. It is invisible to most users but is vital for web accessibility and search engine optimization (SEO).

How do I link to another page?

To link to another page in HTML, you use the a (anchor) tag with the href attribute. The href (Hypertext Reference) attribute specifies the destination of the link

What is the difference between a 'class' and an 'ID' in CSS?

The fundamental difference between a class and an ID in CSS is that an ID is unique to a single element on a page, while a class can be used on multiple elements.

What is the difference between 'margin' and 'padding' in CSS?

The primary difference between margin and padding in CSS is whether the space is applied inside or outside an element's border

When should I use an ol> instead of a ul>?

You should use an ol> (ordered list) when the order of the list items is significant, and a ul> (unordered list) when the order is not important. Both elements use the li> (list item) tag for each item in the list.
